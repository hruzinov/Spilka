# Spilka

A messenger that combines convenience for everyday usage with strict security


## Tech Stack
**Client:** Swift

- **UI:** SwiftUI
- **DM:** SPM
- **Additional:**
  - [SwiftyRSA](https://github.com/TakeScoop/SwiftyRSA) (encryption)

**Server:** Firebase

- Auth
- Firestore Database
